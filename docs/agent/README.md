# Agent Knowledge

本目录维护未来 agent 需要继承的长期上下文。这里不是聊天记录归档，只记录会影响后续判断的事实、决策、术语和交接。

## 文档索引

| 文档 | 维护内容 |
|---|---|
| [context.md](context.md) | 当前项目状态、默认假设、重要边界、下一步 |
| [decisions.md](decisions.md) | 已确认的架构和流程决策 |
| [glossary.md](glossary.md) | 项目术语和命名约定 |
| [handoff.md](handoff.md) | 最近一次工作交接记录 |

## 维护规则

1. 只记录可复用信息，不记录一次性聊天过程。
2. 决策必须包含背景、选择、理由和影响。
3. 交接必须包含已改文件、验证结果、风险和下一步。
4. 如果某条信息已经有权威文档，只链接过去，不复制一份。
