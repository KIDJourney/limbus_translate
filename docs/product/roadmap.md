# Roadmap

本文档维护 AI Workspace 的阶段目标和交付节奏。

## 里程碑

| 阶段 | 目标 | 交付物 | 状态 |
|---|---|---|---|
| v0.1 | 建立面向 agent 的文档知识库骨架 | `AGENTS.md`、`CLAUDE.md`、`docs/`、模板、验证脚本 | 进行中 |
| v0.2 | 将骨架产品化为可复用初始化流程 | 初始化脚本、参数化模板、CI 文档检查 | 未开始 |
| v0.3 | 支持多 agent 工具桥接 | Cursor / Copilot / Gemini 入口生成策略 | 未开始 |
| v0.4 | 支持会话交接和知识沉淀流程 | 交接模板、决策记录流程、变更审计 | 未开始 |

## 当前优先级

1. 保证初版文档结构清晰、可维护、可验证。
2. 保证 agent 入口文件短而稳定，长期知识沉淀在 `docs/`。
3. 保证产品、技术、测试三个维度分离，减少上下文污染。
4. 后续再引入自动化和多工具同步。
